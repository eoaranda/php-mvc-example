# PHP MVC exercise
By Edgar Aranda - 2019

PHP exercise with a mix of different technologies, to demonstrate PHP OOP and MVC design pattern.

# Tech

  - PHP > 5.4 
  - Bootstrap 4
  - JavaScript
  - jQuery
  - Datatables js
  - Chart js
  - Feather js (icons)
  - MySQL - Views & Functions
  - AJAX
  - JSON

Also for the data generation I used:
  - Mockaroo (https://mockaroo.com/)
